8. Rotate String
==
    Given a string and an offset, rotate string by offset. (rotate from left to right)

##8.1 Example
Given "abcdefg".

    offset=0 => "abcdefg"
    offset=1 => "gabcdef"
    offset=2 => "fgabcde"
    offset=3 => "efgabcd"
    
##8.2 Challenge
Rotate in-place with O(1) extra memory.
